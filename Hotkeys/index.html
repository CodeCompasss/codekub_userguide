    Linux Hotkeys: Mastering Your Keyboard

    Linux is keyboard-first, and learning hotkeys is the fastest way to boost productivity.

    1. Navigating the System

    Hotkey Function
    Super + Space Launch app by typing
    Super + A See all apps
    Super + W Close app
    Super + 1/2/3/4 Jump to workspace
    Shift + Super + 1/2/3/4 Move app to workspace
    Alt + 1/2/3/... Jump to dock app
    Ctrl + 1/2/3/... Jump to browser tab

    üí° Tip: Think of Super as your ‚ÄúLinux command key‚Äù.

    2. Tiling Windows

    Hotkey Function
    F11 Full screen
    Shift + F11 Full screen with title bar
    Super + Left/Right Fill left/right half
    Super + Up Fill full screen
    Super + T Open tiling overlay
    Super + Shift + T Open tiling settings
    Super + T W S Fill center vertical
    Super + T Q A Fill left vertical
    Super + T E D Fill right vertical
    Super + T Q Q Fill upper left
    Super + T A A Fill lower left

    üí° Tip: Combine with Super + arrow keys for fast window organization.

    3. Application Windows

    Hotkey Function
    Super + W Close window/application
    Alt + Space Reveal options (Undecorate removes title bar)

    4. Terminal (Zellij)

    Hotkey Function
    Ctrl + G P R New vertical pane
    Ctrl + G P D New horizontal pane
    Alt + Arrow Navigate panes/tabs
    Alt + = / Alt + - Expand / Shrink pane
    Ctrl + G T N New tab
    Ctrl + G T R Rename tab
    Ctrl + G O D Suspend session
    Ctrl + G O W Resume session
    Ctrl + G S E Edit scroll history
    Ctrl + R Reveal command history

    üí° Tip: Zellij turns your terminal into a multi-pane powerhouse.

    5. File Manager

    Hotkey Function
    Ctrl + L Go to path
    Space Preview file
    Backspace Go back one folder

    6. Apple Display Brightness (Optional: ASDControl)

    Hotkey Function
    Ctrl + F1 Decrease brightness
    Ctrl + F2 Increase brightness
    Ctrl + Shift + F2 Max brightness

    7. Neovim (with LazyVim)

    Navigation

    Hotkey Function
    Space Show command options
    Space Space Open file (fuzzy search)
    Space E Toggle sidebar
    Space G G Git controls
    Space S G Search file content
    Ctrl + W W Switch between sidebar & editor
    Ctrl + Left/Right Resize sidebar
    Shift + H / L Move left/right file tab
    Space B D Close file tab

    Sidebar Operations

    Hotkey Function
    A Add new file in parent dir
    Shift + A Add subdir
    D Delete file/dir
    M Move file/dir
    R Rename file/dir
    ? Show help

    üí° Tip: LazyVim turns Neovim into a modern IDE with minimal effort.
