
## fastfetch

**fastfetch** is a **lightweight system information tool for the terminal**.

Think of it as **Neofetch on steroidsâ€”but faster and simpler**.

---

### What It Does

When you run **fastfetch**, it shows:

* **OS and version**
* **Kernel version**
* **Desktop environment / window manager**
* **CPU and GPU info**
* **Memory usage**
* **Shell info**
* **Installed packages**

All of this is displayed **nicely with ASCII logos and colors**.

---

### Why Developers Use fastfetch

* Quickly see **hardware and software info**
* Check **system specs for debugging or sharing**
* Verify **installations and environments**
* Lightweight, runs **instantly** in the terminal

---

### Key Features

* Extremely **fast and lightweight**
* **Highly configurable**: choose what info to show
* Compatible with **Linux, macOS, Windows (WSL)**
* Minimal dependencies
* Optional **ASCII/Unicode logos** for fun
