

## VirtualBox (Run One Computer Inside Another)

Sometimes you want to:

* try a new operating system
* experiment without risk
* learn safely without breaking your main system

**VirtualBox lets you do this.**

---

## What Is VirtualBox?

**VirtualBox** is a **virtualization software**.

It allows you to run a **virtual computer (virtual machine)** inside your real computer.

You can install:

* Linux
* Windows
* other operating systems

inside a window, like an app.
